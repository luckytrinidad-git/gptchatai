=============================
== APP
=============================

Instructions:
To install and run API:
- open gptchatbot-api directory
- run pip install -r requirements.txt
- when ready to run, open terminal and run command, python manage.py runserver

2. To install and run UI:
- open gptchatbot-ui
- run pip install -r requirements.txt
- - when ready to run, open terminal and run command, streamlit run chatbot-ui.py
